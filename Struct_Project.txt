Estruturação do Projeto 

image_classifier_project/
│
├── main.py                 # Código sequencial + Verificações (Arquivos já processados)
├── config.py               # Configurações e argumentos do argparse
├── dataset/
│   └── image_dataset.py    # Classe para carregar, pré-processar e dividir o dataset
├── model/
│   └── classifier.py       # Classe para criação, treinamento e avaliação do modelo
├── utils/
│   ├── logger.py           # Classe para logging
│   └── image_utils.py      # Funções auxiliares (ex: redimensionamento, normalização)
│   └── results_utils.py    # Imprimir os Resultados (Recall, Confusion Matrix, accuracy, F1-score)
├── requirements.txt
└── README.md
